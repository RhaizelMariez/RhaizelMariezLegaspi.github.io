<h2> *Lesson 1 - CSS* </h2>
-> CSS stands for Cascading Style Sheets. <br>
-> It describes how HTML elements are to be displayed on screen, paper, or in other media. It is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes. <br>
-> The 3 layers of a webpage are the behavior layer, presentation layer, and the content layer.  <br><br><br> 


<h2> *Lesson 2 - CSS* </h2>
-> CSS can be applied to HTML/XHTML using three methods: linked, embedded, and inline.<br>
-> Linked Method - CSS is stored in a separate file, instead of directly in the HTML page. <br>
-> Embedded Method - CSS is stored in the header section of the html file. It is defined by <style></style>. <br>
-> Inline Method - CSS is stored directly in the style attributes of the HTML tags. Made outside the html file and uses the <link> element. <br>

<h2> *Lesson 3 - CSS* </h2>
-> HTML browsers that can handle style sheets are not case sensitive. <br>
-> All style declarations must have a colon between the name of the style property and the value/values assigned and must end in a semi-colon. <br>
-> White space must exist where indicated but the user can add extra whitespace to make the code more readable. <br>
-> The different style properties are: fonts, color and backgrounds, text, boxes and layout, lists, and tag classification. <br>
-> The five different possible categories of values are: keywords, length values, percentage values, URL, and color. <br>
-> Display - allows you to control how an element is displayed. The kinds of elements in CSS are block level elements and inline level elements. <br>
-> The types of CSS selectors are the: element selector, ID selector, class selector, universal selector, group selector, and attribute selector. <br>
-> Declarations - describe how these elements (selector) should be displayed and enclosed by curly braces. <br>

<h2> *Lesson 4 - CSS* </h2>
-> Display Property - allows you to control how an element is displayed. <br>
-> Block Level Element - laid out vertically. The p element is an example of this. <br>
-> Inline Level Elements - laid out horizontally. Anchor/link is an example of this. <br>
-> Div Tag - divides the contents of a web page into individual sections. Applied to a block level element. <br>
-> Span Tag - Emphasizes words/sentences within a paragraph. Applied to an inline element. <br>
-> Class - is a user-defined selector. Defined by dots (.). <br>
-> Class Selector - style applied to a particular set of elements. <br>
-> ID Selector - style applied to one element in a page. <br>

<h2> *Lesson 5 - CSS* </h2>
-> Multiple Selectors - when 2 or more elements have the same property and values, a comma is used to separate the selectors and use only one set of properties and values.  <br>
-> Grouping Selectors - you can group selectors and combine it in one CSS file. <br>
-> Nesting Selectors - you can also apply a style for a selector within a selector. You can use the pseudo-class selector and the pseudo-elements. <br>
-> Dimensions - good way of controlling paragraphs, tables, and others. <br>
-> Pseudo-classes are defined by a colon and are not user-defined. <br>
-> The anchor element has four pseudo-classes, namely link, visited, hover and active. <br>
-> Link is for hyperlinks that have not yet been visited. Visited is for visited hyperlinks. Hover is for hyperlinks when the mouse is hovered over them. Active is for hyperlinks when they are clicked. <br>

<h2> *Lesson 7 - FORMATTING LAYOUT WITH CSS* </h2>
-> Box Model - each element on a page exists on its own box wherein each box aligns with other boxes on the page. It consists of margins, borders, padding and the actual content. <br>
-> Element Positioning - position property allows placing an object or element precisely on a page. The four positions schemes to use are: static, relative, absolute and fixed. <br>
-> Offset - allows you to set up the distance of the specified element from another element. <br>
-> Float - allow you to control where a specified block element will be located or placed on the page and where the other elements will be positioned in relation to the said block. <br>
-> Wrapping Text around an Element - can be done using the float property in CSS. <br>
-> Background-repeat property - indicates if the background image is to be tiled all throughout (repeat value), repeated horizontally only (repeat-x value), repeated vertically only (repeat-y value) or will not be repeated nor tiled (no-repeated value). <br>
-> Background-position property - indicates where the background image will be placed. <br>
-> Background property - is a shorthand notation for all the other properties (background-color, background-image, background-repeat, background-attachment and background-position). <br>
-> Background image property allows you to add a background image to the entire Web page or to a single element. <br>
-> Margins - if you place only one value on the margin:<value>, that value will be applied to all sides of the specified element. <br>
